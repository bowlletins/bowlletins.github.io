## bowlletins.github.io

Bow-lletins is a web application designed for UH Mānoa students that centralizes campus announcements, events, and opportunities into one easy-to-use digital bulletin board. Instead of relying on scattered flyers, emails, and word-of-mouth, users can browse, search, and interact with posts in a single platform.

This repository contains the documentation and informational content for the Bow-lletins system.

Visit the site at: https://bowlletins.github.io
